# Lovart Prompt Translator

A Tampermonkey userscript for translating, comparing, editing, and refilling Chinese/English prompts in Lovart input boxes.

## Install

Install the script from the GitHub Raw URL:

https://raw.githubusercontent.com/wsbjj/lovart-prompt-translator/main/lovart-prompt-translator.user.js

## Usage

1. Install Tampermonkey or another compatible userscript manager.
2. Open the install URL above.
3. Confirm the userscript installation.
4. Open `https://lovart.ai/` and configure your translation provider from the script settings.

## Files

- `lovart-prompt-translator.user.js` - the userscript for installation.
- `lovart-prompt-translator.test.js` - lightweight source checks.
- `preview-settings.html` - local settings preview page.
